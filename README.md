# YOOF.ai — Digital Playground for Self-Discovery

Welcome to the official GitHub repository for the YOOF.ai homepage.  
YOOF is a UK-based platform built to spark curiosity and exploration through AI.

## 🚀 About

This is a static site built with HTML and CSS and hosted via GitHub Pages.  
It serves as the entry point into the YOOF.ai directory — a growing resource for tools, experiments, and projects that inspire personal growth in both youth and adults.

## ✨ Features

- Responsive layout (mobile-friendly)
- UK English language and style
- Accessible, playful design
- Simple to maintain and expand

## 🔧 Usage

1. Clone this repository  
   `git clone https://github.com/YOUR-USERNAME/yoof-homepage.git`

2. Open `index.html` in your browser or edit with your preferred code editor

3. To publish:
   - Push changes to the `main` branch
   - Enable GitHub Pages in repo settings (choose `main` branch as source)
